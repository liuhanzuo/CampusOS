# 体育场馆预约系统 - 快速开始指南

## 🚀 5分钟快速上手

### 1. 安装依赖

```bash
cd apps/thu-ai-assistant
npm install selenium-webdriver
```

### 2. 启动命令行工具

```bash
npm run sports
```

### 3. 按照提示操作

1. 输入你的清华学号和密码
2. 如果需要二次认证：
   - 选择验证方式（企业微信推荐）
   - 查看企业微信获取验证码
   - 输入6位验证码
   - 选择是否信任设备（推荐选择"是"）
3. 登录成功后，选择操作：
   - 输入 `1` 查询场馆
   - 输入 `2` 预约场地
   - 输入 `3` 查看所有场馆

## 🔧 常见问题

### Q: 登录失败怎么办？

A: 删除 `.cookies` 目录后重新登录：
```bash
rm -rf apps/thu-ai-assistant/.cookies
npm run sports
```

### Q: 找不到ChromeDriver？

A: 程序会自动下载，如果失败请手动安装：
- 访问：https://chromedriver.chromium.org/
- 下载与你Chrome版本匹配的Driver

### Q: 验证码输入错误？

A: 验证码有效期很短，请及时输入。可以重新获取验证码。
